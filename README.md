# Frontend Bootcamp

A modern, responsive landing page designed to promote a web development bootcamp. It features sections highlighting what students will learn, instructor information, and a newsletter signup, all styled with Bootstrap for a clean and professional look. The main focus is to introduce new developers to essential frontend technologies and encourage them to enroll in the program.

## Features

- **Dynamic Landing Section:** Eye-catching introduction that promotes the Frontend Bootcamp and its benefits.
- **Instructor Showcase:** Spotlights the instructors with details about their expertise.
- **Newsletter Signup:** Integrated input form for visitors to subscribe to news and updates.
- **Responsive Layout:** Fully optimized for desktops, tablets, and mobile devices.
- **Modern User Experience:** Clean UI design with intuitive navigation and visually appealing components.
- **Bootstrap & Icon Integration:** Utilizes Bootstrap and icon libraries for consistency and style.
- **Interactive Map Feature:** Embedded Leaflet map to showcase locations or provide geographical context.

## Languages Used

- **HTML:** Provides the foundational structure and semantic layout of the webpages.
- **CSS:** Handles all aspects of styling, layout adjustments, and responsive design.

## Technologies & Libraries Used

- **Bootstrap:** Responsive layout, pre-designed components, and streamlined UI development.
- **Bootstrap Icons:** Vector icons and symbols used throughout the site.
- **Leaflet:** Interactive map display embedded into the website.
